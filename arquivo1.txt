6. Buscar Post por ID
GET /posts/:postId

Headers: Authorization: Bearer <TOKEN_JWT>

Esperado: 200 OK + post

Erro: 404 (post não encontrado)

7. Deletar Post
DELETE /delete/post

Headers: Authorization: Bearer <TOKEN_JWT>

Body (JSON):

json
Copiar
Editar
{
  "postId": 1
}
Esperado: 200 OK

Erro: 401 (token inválido)

8. Criar Comentário
POST /posts/:postId/comments

Headers: Authorization: Bearer <TOKEN_JWT>

Body (JSON):

json
Copiar
Editar
{
  "content": "Comentário aqui",
  "userId": 1,
  "postId": 1
}
Esperado: 201 Created

9. Listar Comentários
GET /posts/:postId/comments

Headers: Authorization: Bearer <TOKEN_JWT>

Esperado: 200 OK + lista de comentários

10. Deletar Comentário
DELETE /feed/posts/:postId/comments/:commentId

Headers: Authorization: Bearer <TOKEN_JWT>

Esperado: 200 OK

11. Curtir ou Descurtir Post
POST /posts/:postId/like

Headers: Authorization: Bearer <TOKEN_JWT>

Body (JSON):

json
Copiar
Editar
{
  "userId": 1,
  "postId": 1
}
Esperado: 200 OK

12. Ver Meu Perfil
GET /me

Headers: Authorization: Bearer <TOKEN_JWT>

Esperado: 200 OK + dados do usuário logado

13. Ver Perfil por Nick
GET /perfil/:usernick

Headers: Authorization: Bearer <TOKEN_JWT>

Esperado: 200 OK + dados do perfil buscado

14. Trocar Avatar
PATCH /troca/avatar/:avatar

Headers: Authorization: Bearer <TOKEN_JWT>

Body (JSON):

json
Copiar
Editar
{
  "profilePicture": "https://exemplo.com/avatar.jpg"
}
Esperado: 200 OK
